# E-Commerce Product Service

## Overview
`ecommerce-product-service` is a microservice responsible for managing product-related operations in the E-Commerce Microservices Application.

It handles product creation, updates, deletion, and retrieval.  
This service communicates with the API Gateway and registers itself with Eureka Server.

---

## Features

- Add new products
- Update existing products
- Delete products
- View all products
- View product by ID
- Service discovery with Eureka

---

## Tech Stack

- Java 
- Spring Boot
- Spring Data JPA
- MySQL
- Eureka Client
- REST APIs
